# Carousel Creation SOP

1. **Content Enhancement & Source Adherence**: If given a content plan or idea, analyze it and enhance it to make Instagram attention-seeking content. **CRITICAL**: When provided a visual reference image, ONLY use it for structural design and layout inspiration. Do NOT copy the actual text/content from the reference image. The content must always be strictly based on the provided website/source material.
2. **Design Skill Requirement (UI/UX Pro Max)**: When making any design, ALWAYS use the `ui ux pro max` rules. Ensure generous padding, proper flexbox alignment (avoid absolute positioning that causes overlap), and prevent text overlapping with bottom elements.
3. **Carousel Content Framework**: Always follow the `carousel 101` file when structuring and writing carousel content. Focus on the Hook -> Rehook -> Mirror -> Reveal -> CTA narrative arc.
4. **HTML Layout for Screenshotting**: When generating the final HTML file, output a **vertical, scrollable feed** (like a mobile webpage) with an 80px gap between slides, rather than a horizontal track. Ensure a fixed 4:5 aspect ratio (e.g., 540x675px). Disable hover scaling/animations so the slides remain perfectly still for the user to screenshot easily.
5. **Strict Visual Consistency (Light Theme Aesthetic)**: Stick to a single, consistent modern font family (e.g., Plus Jakarta Sans). Maintain strict color token consistency: Backgrounds must be light (white or subtle soft colors) and Primary Text must be Dark Blue (e.g., #0F172A). For highlights, use an amazing combined color font (e.g., a striking blue-to-orange gradient) consistently across all slides instead of mixing different single text colors.
6. **No Watermarks & Trademarks**: Do NOT include any brand names, logos (e.g., "Rocket Carousel Maker"), website URLs, or @handles in the slide design. Leave those areas blank or use them purely for structural aesthetics.
7. **Attention-Seeking Visuals**: Always incorporate rich visual elements (e.g., glassmorphic cards, soft background blur blobs, custom notice boxes, and bold icons) to visually represent the text and aggressively grab user attention, avoiding plain text-heavy slides.
8. **Reference Decoding**: If a reference carousel is provided, decode its visual structure, layout, and UI patterns, then create a new one incorporating our design system and elements. Do not copy the reference's stock images if they conflict with our premium aesthetic.
9. **Token Conservation**: Always be token conservative and use limited tokens as possible.
10. **SOP Adherence**: Always keep this SOP in mind and follow these steps meticulously for every future carousel creation task to avoid repeating revisions.
